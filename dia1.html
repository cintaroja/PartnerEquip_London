Control Automation

  • Automate evidence collection and generate reports
  • Common controls - Created and maintained by AWS Sec Assurance Services, LLC. 13 control domains
  • AWS Artifact provides on-demand access to AWS security and compliance reports and select online agreements (NEW: Automate report consumption)

Meeting Your RPO and RTO Goals Using AWS Resilience Hub

Cristian Critelli
13:00 - 13:45

What is Resilience?

Refers to the ability of workloads to respond to and quickly recover from failure. It is good to have an objective (4 9s, 5 9s...)

AWS Resilience Hub

Service that provides customers a central place to define, evaluate, validate and track the resilience of their apps on AWS

Disruption Types

  • App data
  • Infra
  • Availability zone
  • Region

Supported Resources

  • Compute
  • Networking
  • Database
  • Storage
  • Other

To Use It

  1. Define resilience targets
  2. Add app
  3. Assess app resilience
  4. Implement remediations
  5. Test app resilience (GOTO 3)
  6. Track resilience posture

Pricing

Resilience Hub free for 6 months and then cheap (~$15/month)

Demo Highlights

  • Se puede adaptar la herramienta (res hub) para que mire a tu app "como quieres"
  • Si hago backups sin la herramienta de backups de AWS puedo configurar res hub para que mire con una lambda si tengo los backups hechos
  • Para empezar vamos a Resilience Hub Console

  • Free tier

    • Con región específica

Savings Plans

Cost Explorer and Recommendations
